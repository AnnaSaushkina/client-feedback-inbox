## 🚀 Запуск проекта

### 2️⃣ Запустить сервер (порт 3000)

```bash
node server/server.js
```

Проверка:

```
http://localhost:3000/api/tasks
```

---

### 3️⃣ Запустить фронтенд (Vite, порт 5173)

```bash
npm run dev
```

Открыть:

```
http://localhost:5173
```

---

## ⚙️ Важно

В `vite.config.js` должен быть proxy:

```js
server: {
  proxy: {
    "/api": "http://localhost:3000",
    "/uploads": "http://localhost:3000"
  }
}
```

## 📂 Хранение данных

- Задачи → `server/db.json`
- Загруженные файлы → `server/uploads/`

---

## 🛑 Если ошибка ECONNREFUSED

Проверь, что сервер (`node server/server.js`) запущен.
