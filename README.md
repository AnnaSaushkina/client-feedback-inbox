# Приложение для управления задачами

## Что умеет

- Создавать задачи через модальное окно
- Переключать задачи между Активными и Выполненными
- Удалять задачи с подтверждением
- Сохранять всё в localStorage
- Адаптироваться под мобильный и десктопный экран

## Стек

React 18 · TypeScript · Ant Design 5 · Vite · localStorage

## Запуск

```bash
pnpm install
pnpm dev
```

## Что планирую добавить

- Редактирование задач
- Фильтрацию по тексту и статусу
- Приоритеты и дедлайны
