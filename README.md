### Доска учета комментариев

Внутренний инструмент команды разработки для фиксации и управления клиентскими комментариями из Telegram-чатов.

Приложение преобразует поток сообщений в структурированные задачи с управляемым workflow.

#### Что делает система

- Создание задачи (скриншот комментария + описание для разработчика)
- Канбан-доска по статусам
- Приоритеты и назначение исполнителей
- Фильтрация и поиск
- Подсветка просроченных задач
- Карточка задачи с историей изменений статусов

#### Предполагаемый стек

React + TypeScript + Vite + Ant Design + MobX + Vitest

#### Цель проекта

Разработать полноценное продуктовое приложение с прозрачным процессом обработки клиентских доработок и внедрить его в рабочий процесс команды.
