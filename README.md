# Task Board

SPA для управления задачами команды на основе клиентских комментариев.

## Что умеет

- Создавать и редактировать задачи с тикетом, дедлайном, исполнителем и статусом
- Статусы: Высокий / Средний / Низкий приоритет · Ждём коммента (серая задача)
- Прикреплять скриншоты через Ctrl+V или загрузку файла
- Сортировать по срочности и приоритету автоматически
- Подсвечивать дедлайн: просрочен / горит / ок
- Копировать ежедневный отчёт одной кнопкой (задачи «Ждём коммента» не попадают в отчёт)
- Работать с API или в офлайн-режиме (localStorage)

## Стек

**Фронтенд:** React 19 · TypeScript · Ant Design 6 · Vite

**Бэкенд:** Express.js · SQLite · Node.js

**Деплой:** VPS · Nginx · PM2

## Запуск

```bash
pnpm install
pnpm dev
```

Бэкенд запускается отдельно, URL задаётся через `VITE_API_URL` в `.env`.

## Тесты

```bash
pnpm test
```
