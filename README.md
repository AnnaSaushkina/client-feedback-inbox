# Task Board

SPA-приложение для управления задачами разработки на основе клиентских комментариев.

## Что умеет

- Создавать задачи с названием, описанием, номером тикета, дедлайном, приоритетом и исполнителем
- Прикреплять скриншоты через Ctrl+V
- Открывать карточку задачи по клику и редактировать её
- Переключать задачи между активными и выполненными
- Удалять задачи с подтверждением
- Сортировать по срочности и приоритету автоматически
- Подсвечивать дедлайн цветом: просрочен / горит / ок
- Копировать ежедневный отчёт одной кнопкой
- Хранить данные в localStorage

## Стек

React 18 · TypeScript · Ant Design 5 · Vite · Vitest

## Запуск

```bash
pnpm install
pnpm dev
```

Приложение доступно на `http://localhost:5173`.

## Тесты

```bash
pnpm test
```
