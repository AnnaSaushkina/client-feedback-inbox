# Task Board — Система управления задачами

SPA-приложение для централизованного управления задачами разработки на основе клиентских комментариев.

## Что умеет

- Создавать задачи с названием, описанием, номером тикета, дедлайном, приоритетом и исполнителем
- Прикреплять скриншоты через Ctrl+V — можно несколько
- Показывать превью скриншота и метаданные прямо в реестре задач
- Открывать полную карточку задачи по клику
- Редактировать задачу из карточки
- Переключать задачи между «Активными» и «Выполненными»
- Удалять задачи с подтверждением
- Фильтровать по тексту, приоритету и исполнителю
- Автоматически сортировать задачи по срочности и приоритету
- Подсвечивать дедлайн цветом — просрочен / горит / ок
- Копировать ежедневный отчёт одной кнопкой
- Сохранять все данные в базе данных на сервере

## Стек

**Фронт:** React 18 · TypeScript · Ant Design 5 · Vite · Vitest  
**Бек:** Express.js · SQLite
**Деплой:** GitHub Pages (фронт) · VPS (бек)

## Запуск

**Фронт:**

```bash
pnpm install
pnpm dev
```

**Бек:**

```bash
npm install
npm run dev
```

Фронт доступен на `http://localhost:5173`, бек на `http://localhost:3000`.

## Переменные окружения

Создай `.env` в корне фронта:

```
VITE_API_URL=http://localhost:3000
```

## Тесты
